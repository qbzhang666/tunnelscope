# Assets directory

Place the following binary files here. They are excluded from this text-only
scaffold so you can drop in your own imagery.

## Expected files

- `logo.png` — Your project or Transurban logo (optional, used in sidebar
  if present). Recommended size: 200x200px.

- `tunnel_cross_section.png` — A reference cross-section illustration of
  the tunnel showing superstructure / central structure / substructure
  zones and lane markings. Used on the overview page (optional).

- `example_rgb.jpg` — Sample RGB inspection image with visible defects,
  for the CV → COBie demo page.

- `example_thermal.tiff` — Sample thermal IR image at the same location.

- `example_gpr_profile.png` — Sample GPR radargram showing a void
  reflection.

- `example_pointcloud.ply` — Small sample point cloud (optional, for
  RGBD demos).

The app works without these — they are only used for illustrative
context in the UI.
